# JavaScriptでモーダルを作成する
## 目的
- webpackとgulpの設定について学ぶ
- jQueryを使わずに処理をする

## Local development environment

ローカル開発の基本となる環境一式。

- nodenv
- EditorConfig
- ESLint (エディタ)
- StyleLint (エディタ)
のインストールを推奨。

### Node version
14.8.0

### Install
```
$ npm install
```

### Development
```
$ npm run dev
```

### Production
```
$ npm run prod
```
